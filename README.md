# IBM Data Science Capstone

## SpaceX Falcon 9 First Stage Landing Prediction

### Project Description
This project predicts the successful landing of the SpaceX Falcon 9 first stage using machine learning techniques.

### Technologies Used
- Python
- Pandas
- SQL
- Matplotlib
- Seaborn
- Folium
- Plotly Dash
- Scikit-learn

### Repository Contents

- Data Collection API Notebook
- Data Collection using Web Scraping
- Data Wrangling
- Exploratory Data Analysis (EDA)
- SQL Analysis
- Folium Map
- Plotly Dashboard
- Machine Learning Prediction
- Final Presentation

### Best Model

Support Vector Machine (SVM)

Accuracy: **88%**
